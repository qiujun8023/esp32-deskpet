# esp32-deskpet

基于 ESP32 的桌面小机器人。OLED 显示动态眼睛表情，手机连接 WiFi 热点后通过虚拟摇杆实时控制移动，也可切换自主漫游模式让它自己乱跑。

---

## 硬件

| 配件 | 规格 |
|------|------|
| 主控 | ESP32-C3 Super Mini |
| 屏幕 | 0.96" OLED SSD1306，128×64，I2C |
| 电机 | 1812-N20 蜗轮蜗杆电机 × 2 |
| 驱动 | 迷你 N298N 两路电机驱动 |
| 电源 | 14250 锂电池（3.7V，Type-C 充电）|

## 引脚

| GPIO | 功能 |
|------|------|
| 8 / 9 | OLED SDA / SCL |
| 0 / 1 | 左轮前进 / 后退 |
| 2 / 3 | 右轮前进 / 后退 |
| 10 | N298N STBY |

---

## 编译烧录

```bash
idf.py set-target esp32c3
idf.py build
idf.py flash monitor
```

## 使用

上电后手机连接 WiFi **`ESP32-DeskPet`**（无密码），浏览器访问 `10.10.10.10` 打开控制页面。
